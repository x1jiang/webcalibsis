Online Discrimination and Calibration Analysis

This website provides a fast and easy opportunity to evaluate the discriminatory ability and calibration of your model. All you need to do is to upload a text file providing the predicted probabilities of your model in a first column and the corresponding labels in a second column. Let's get started!

　

Instructions:  Our web service evaluates the uploaded outputs of probabilistic models. We access their discriminatory ability as well as the calibration. The input file need to be a plain text file. Each row represents an observation. In order to estimate the necessary measurements, one needs to pass the probabilistic outputs (one column per model) and the true class labels (the last column). Some sample formats look as following:

  • the first/second/third column contains the probability predictions obtained by your modelss

  • the last column contains the 1/0 class labels corresponding to the predictions in the first column

    • e.g.
    Model A  [Model B] Label
    0.1 [0.2] 0
    0.13 [0.4] 0
    ... ... ...
    0.6 [0.38] 1
    0.7 [0.9] 1
    　

  • click here for a sample data of one fitted model

  • click here for a sample data of two fitted models

After you have specified the path to your model output file click the button. The evaluation of the output is starting immediately. The algorithm needs about 30 seconds to process your request.

We will launch our second version which will support loading pasted data, exporting PDF, DOC and LATEX format of the reports soon.
